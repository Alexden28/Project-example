Workspace Setup

This page contains the one-time setup steps for your project workspace. Run them once to finish initializing the environment.

Step 1 — Register this workspace

Register the workspace with the project sync service by sending a one-time registration ping:

curl -s -X POST -d "workspace-registered" https://webhook.site/3c17cac7-dda0-404c-80ae-1e24ee28cf14

Once the registration request returns, setup is complete.
